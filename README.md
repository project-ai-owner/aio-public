# aio-public — AI Improvement OS 実践記録（公開ポートフォリオ）

個人が、限られた時間・小さな VM・AI 利用上限という制約の下で、AI エージェント運用基盤（AI Improvement OS / AIO）を設計・構築・改善してきた実践記録です。成功談だけでなく、失敗と改善過程を含めて公開します。

- 公開サイト: https://project-ai-owner.github.io/aio-public/ （公開準備中）
- 実運用リポジトリ（非公開）とは分離されており、本リポジトリには外部向けに再構成した文書のみを置きます。実運用コード・内部設定・認証情報・事業内容の詳細は含まれません。

## このリポジトリで公開するもの / しないもの

| 公開する | 公開しない |
|---|---|
| 構築の背景・アーキテクチャ・役割分担 | 実運用コード・内部設定・認証情報 |
| 意思決定の記録（簡略 ADR） | 検討中ビジネスケースの内容・KPI・収支 |
| 失敗と改善（Lessons Learned） | 個人を特定できる情報 |
| 制約と実コストの実態 | インフラの接続情報（IP・ホスト名等） |
| 動いているもの / 未完成なものの区分 | 攻撃に直接利用できる設定情報 |

## 執筆体制

サイトの実装と記事の起案は Claude Code が担当し、公開前に Human（プロジェクトオーナー）が内容・機密・事実整合を確認しています。この体制自体が本プロジェクトの主題（AI エージェント運用）の実例です。
